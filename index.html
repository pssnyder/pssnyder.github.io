About This Portfolio

This site represents a unique approach to technical documentation and project showcasing. Rather than static project descriptions, you'll find living documents that capture both the technical implementation details and the thought processes behind each project.

Each entry combines narrative context (the "why") with data analysis and code (the "what" and "how"), providing a complete picture of the problem-solving process. Projects range from chess engine development to data science explorations, all documented with the same level of analytical rigor.
